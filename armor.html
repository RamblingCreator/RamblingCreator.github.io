Armor

Armor was one of the first things I made when I was learning leatherworking. In 2020, I took an online class on creating armor. This is when I made the blue armor. I decided that I didn't really like how it turned out, so I started to work on another set and I've added to it each summer since. I first made one gauntlet, then one pauldron. Next summer, I made the other side for each piece. My plans for the future are to make some sort of breastplate.
